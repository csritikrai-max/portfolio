## 🌐 Personal Portfolio – Ritik Rai

Welcome to my personal portfolio website!

* ✨ Modern and responsive UI
* 🧑‍💻 About Me section
* 📄 Resume download option
* 📬 Contact form with backend integration
* 🗄️ Data storage using SQLite database
